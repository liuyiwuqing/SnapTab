## Detailed Description

### Chinese

SnapTab 是一款专注效率的 Chrome 截图扩展。  
你可以在网页中快速完成全页面、区域、标签页和元素截图，并在内置编辑器中进行裁剪、标注和文字添加。  
截图完成后，支持应用多种套壳样式（设备壳、浏览器壳、边框壳），导出为 PNG/JPG、本地保存、复制到剪贴板，并通过浏览器分享能力快速分发。  

**核心功能**

- 四种截图模式：全页/区域/标签页/元素
- 基础编辑：裁剪、矩形、箭头、画笔、文字、撤销/重做
- 套壳系统：12 套内置模板 + 用户自定义模板上传
- 导出与分享：PNG/JPG、本地下载、剪贴板复制、Web Share、二维码分享路径
- 隐私优先：截图内容默认仅在本地处理

### English

SnapTab is a productivity-first screenshot extension for Chrome.  
Capture full page, region, visible tab, or selected element, then edit instantly with crop, annotation, and text tools.  
Apply polished frames (device, browser, and border styles), export as PNG/JPG, save locally, copy to clipboard, and share using browser-native workflows.  

**Key Features**

- 4 capture modes: full page, region, tab, and element
- Built-in editor: crop, rectangle, arrow, pen, text, undo/redo
- Frame system: 12 built-in styles + custom template upload
- Export & share: PNG/JPG, download, clipboard copy, Web Share, QR-based sharing path
- Privacy-first: screenshot processing stays local by default
